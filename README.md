## Installation

-   From PyPi

    ```bash
    pip install snakify
    ```

-   From Source :

    ```bash
    git clone https://github.com/bhatishan2003/snakify.git
    cd snakify
    pip install .
    ```

## Usage

Run the following command in the terminal to play the game.

```bash
snakify
```
